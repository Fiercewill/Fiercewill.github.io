Git&GitHub是什么？

Git是一个免费和开源的分布式版本控制系统（VCS），旨在快速高效地处理从小型项目到大型项目的所有项目。

版本控制系统（VCS）可以帮你保存文件的所有修改记录并使用版本号进行区分，可以随时浏览历史版本，将该错的文件还原到指定的版本，还可以对比不同版本的文件差异，起到恢复和保护的作用。而Github是主流i的代码托管平台，可以理解为一个存放和管理代码的网盘，大家可以把自己的的代码传上去，进行共享和维护，当然也可以从平台下载代码。

Git是工具，是抓手；而GitHub是平台，两者没有必然的联系。

为什么要用Git&GitHub？

他们是为协作开发而生的。大多数公司都在用他们来管理项目，只要我们想找工作就不得不学。

准备开发个人项目前，可以学习Git，这样就可以把做的所有项目放到GitHub去管理。
