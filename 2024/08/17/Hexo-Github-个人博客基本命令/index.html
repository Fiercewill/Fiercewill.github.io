前言

时间来到2024年8月17日，距离这个博客搭建好已经有一年多了，也已经很久没有在这上面更新文章了，原因如下：

（1）个人博客需要是搭建在github上的，需要借助外网才能访问，但是在连接到外网的情况下，观看博客还是不流畅，体验感不好，一般我们在查看笔记的时候都需要高效率，不然等到文章打开，等着连看博客的心思都没有了。

（2）其次虽然看这些文章也可以在电脑的文件中观看，但是文件夹里的文件内容就很乱，条理不清楚，所以阅读的体验感也不好。

基于以上原因，就不是很乐意以此种方式继续记笔记了。

但是话说回来，这个博客都搭建好了，而且也准备在简历上贴上这个博客（因为csdn博客的链接太长了，这个博客的网址还有github显得高级一些）所以还是要讲得出来这个博客是怎么搭建的，了解写博客的基本命令和操作。
